# coin-monitor
Repository for tauri application distribute tutorial

## Installation

1. Download the installer for your operating system:

> [!IMPORTANT]  
> Linux is not currently supported.

[![MacOS](https://img.shields.io/badge/-MacOS-0078D6?style=for-the-badge&logo=apple)](https://github.com/jhk0530/coin-monitor/releases/download/v0.0.1/coin-monitor_0.1.0_aarch64.dmg)

2. Run the installer and follow the prompts.
3. Launch `coin-monitor` from your applications menu.
